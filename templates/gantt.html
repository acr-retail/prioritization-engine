How Gantt Scheduling Works
Bar Position & Length
  • Start Date determines where the bar begins
  • End Date determines where the bar ends
  • If no End Date is set, the bar length defaults to the Level of Effort duration
  • If no Start Date is set, the bar starts at today
Constraints
  • Level of Effort sets the minimum bar duration — the bar cannot be shorter than this
  • Deadline (red flag ⚑) is a hard limit — the bar cannot extend past this date
  • Effort mapping: <10 Hrs = 1 day, 10-40 Hrs = 5 days, 41-100 Hrs = 12 days, >100 Hrs = 25 days
Grouping & Filtering
  • Group by — organize tasks by Project, Assignee, Customer, or Status. Default is Project. Your selection is remembered.
  • Click a group header — for Projects, opens the project detail panel. For other groupings, filters to that value.
  • Child tasks are indented under their parent task when grouped by Project
  • Use the filter dropdowns (Project, Customer, Assignee, Status) to narrow the view
Zoom Levels
  • Daily — one column per day, 3-month range
  • Weekly — one column per week, ~3-month range
  • Monthly — one column per month, ~14-month range
  • Quarterly — one column per quarter, ~2-year range
Interactions
  • Drag a bar horizontally to change the Start and End dates together
  • Drag the right edge to resize (change End Date only, respects minimum effort duration)
  • Click a bar or task name to open the task editor panel with all fields and conversation
  • All changes save to Odoo automatically
  • Every panel includes an "Open in Odoo →" link
Colors
  • Blue — assigned to a developer
  • Red — unassigned
  • Red vertical line — today
  • ⚑ Red flag — deadline marker (bar cannot extend past this)
